# 时间大屏 · Time Web

一个纯前端的时间大屏与日历日程展示页，适合在电视/会议室/办公室大屏常驻显示当前时间、日期、日历与当天日程。项目包含大屏展示页与设置页，所有配置保存于浏览器本地存储（LocalStorage/Cookie），无需后端。

## 功能亮点
- 大屏时钟：支持 24 小时制/12 小时制、显示秒、全屏显示
- 日历视图：月视图、周起始可配置、当天高亮、快速回到“今天”
- 日程面板：按日期查看日程，支持折叠/展开
- 日历订阅：支持导入 .ics 订阅（浏览器端抓取）
- 手动日程：在设置页快速添加/删除临时事项
- 自适应布局：适配不同分辨率屏幕与横竖屏

## 应用场景
- 会议室/办公区大屏的时间与日程展示
- 前台/大厅的实时日期与日历展示
- 个人工作区的“今日安排”看板
- 课堂/培训教室的课程安排提示屏

## Demo 地址
- 线上演示：https://zepengw.github.io/time-web/
- 本地演示：直接用浏览器打开 `index.html` 即可

## 使用方法
1. 直接打开 `index.html` 进入大屏
2. 点击“设置”进入 `settings.html` 配置时区、周起始、日历订阅与手动日程
3. 保存后返回大屏，即可看到更新

## 备注
- 时钟基于 Time.is
- 订阅源若不支持跨域访问，浏览器会提示同步失败
- 配置量过大时可能超过 Cookie 限制，请减少订阅或手动日程
